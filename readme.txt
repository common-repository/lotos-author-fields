This plugin adds additional fields in user info.

== Description ==

Adds additional meta fields for User Profile and link on author page in widget "Recent Comments"


If you liked my plugin, please rate it.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Users->Your Profile, add Contact Info.
